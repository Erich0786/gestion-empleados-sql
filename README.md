# Sistema de Gestión de Empleados (SQL)

Este proyecto contiene la estuctura inicial para una base de datos para la administración de personal.

## Estructura de las Tablas
* Empleados: Registra datos personales (Nombre, rut único y apellidos).
* Sueldos: Almacena la información financiera vinculada a cada empleado mediante una llave foránea (FK).

## Cómo usar
1. Abrir MySQL Workbench.
2. Ejecutar el script `script_creacion.sql`.
3. Refrescar los schemas para ver `tabla_empresa`.
